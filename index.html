Project : 1
Share Analysis - Web App

ShareAnalysis is a sophisticated stock research for share market investment on ASX and global stocks. It has everything to make smarter decisions for share portfolio. It provides estimate on portfolio for future income, generate filters as per user requirement, and send email alerts when user need to place a buy or sell trade. ShareAnalysis compare tool compares the earnings, dividends, debt, return on equity, cash flows and future growth forecasts for up to 15 stocks side-by-side.

An application which could be used as an advertisement or promotion application. It scans for all current remote devices. Then send the bulk of predefined files to all searched devices. Here, files can be text file or image, video or any kind of file. The admin user is able to change these settings.
